## 这节课你要拿到什么

- 一份可以直接执行的 7 天计划
- 每天一个明确交付物
- 一周结束后：3 个模板 + 1 个工作流 + 1 个知识库

## 总原则

这 7 天不追热点、不比模型、不换工具。只做一件事：用 AI 完成真实任务。

## Day 1：盘点任务（30 分钟）

列出你最近一周做过的 10 个重复任务，用第1课学的四象限分类。

| 任务 | 耗时 | 是否重复 | 有固定输入吗 | 出错代价 | 分类 |
| --- | --- | --- | --- | --- | --- |
| 周报整理 | 40 分钟 | 是 | 是 | 低 | A |
| 会议纪要 | 30 分钟 | 是 | 是 | 低 | A |
| 客户报价 | 25 分钟 | 是 | 是 | 高 | B |

从中选 1 个 A 类任务作为本周主线。

✅ 交付物：一张标好分类的任务清单，选定 1 个起步任务。

## Day 2：写模板（30 分钟）

用第2课学的"角色+目标+约束+输出"结构，给选定的任务写一个提示词模板。

以周报为例：

```text
你是我的工作助手。

目标：根据任务清单生成周报。

约束：
- 500 字以内
- 分"本周完成""下周计划""风险提示"三部分
- 语气简洁专业

输入：
[粘贴本周任务清单]

输出：Markdown 格式
```

写完后打开你的对话助手（豆包/元宝/通义千问/Kimi），把模板和真实材料粘贴进去跑一次。

✅ 交付物：1 个提示词模板 + 1 次真实输出。

## Day 3：跑真实任务（30 分钟）

用昨天的模板完成一次真实工作交付（不是练习，是真的发出去/提交的那种）。

跑完记录 3 件事：

| 记录项 | 你的答案 |
| --- | --- |
| AI 输出哪里好？ | |
| 哪里需要手动改？ | |
| 模板哪里要调整？ | |

根据记录修改模板，保存为 v2。

✅ 交付物：1 次真实交付 + 模板 v2。

## Day 4：配工具栈（30 分钟）

按第4课的方法，配齐 4 个角色的工具。

推荐新手直接用组合 A：

| 角色 | 工具 |
| --- | --- |
| 对话助手 | 豆包 / 元宝 / 通义千问 |
| 信息核查 | 秘塔 AI 搜索 + 官网验证 |
| 知识沉淀 | 飞书文档 |
| 流程串联 | 飞书多维表格 |

配完后，把 Day 2-3 写的模板存进飞书文档的"模板库"文件夹。

✅ 交付物：4 个工具注册完成 + 模板已存入知识库。

## Day 5：再跑两次（30 分钟）

用同一个模板再做 2 次真实任务。重点观察：

- 哪一步最慢？
- 是不是每次都要改同一个地方？
- 模板还缺什么？

根据观察更新模板为 v3。

✅ 交付物：累计 3 次真实交付 + 模板 v3。

## Day 6：建知识库（30 分钟）

按第5课的方法，在飞书文档里建好完整的知识库结构：

```
AI 工作台/
├── 📁 模板库/        ← 存 Day 2-5 写的模板
├── 📁 流程库/        ← 存 Day 4 的工作流 SOP
├── 📁 成功案例/      ← 存效果好的输出
└── 📁 失败复盘/      ← 存翻车记录
```

把本周所有模板和记录归档进去。再写 1 条失败复盘（这周肯定有不满意的输出）。

✅ 交付物：知识库建好 + 至少 3 个模板归档 + 1 条失败复盘。

## Day 7：复盘 + 启动第二个任务（30 分钟）

用这张表复盘整周：

| 问题 | 你的答案 |
| --- | --- |
| 哪个任务最适合 AI？ | |
| 哪个模板最稳定？ | |
| 哪一步必须人工把关？ | |
| 下周准备优化什么？ | |
| 下一个要做的任务是？ | |

然后从 Day 1 的任务清单里选第 2 个 A 类任务，重复 Day 2-6 的流程。

✅ 交付物：1 份复盘记录 + 选定下周任务。

## 案例：小张的 7 天记录

小张是产品经理，每周写周报、整理需求文档、做竞品分析。

| 天 | 做了什么 | 交付物 |
| --- | --- | --- |
| Day 1 | 列了 8 个任务，选了"周报"作为起步 | 任务清单 |
| Day 2 | 写了周报模板 v1，用豆包跑了一次 | 模板 v1 + 1 次输出 |
| Day 3 | 用模板写了真实周报，发给领导 | 真实交付 + 模板 v2 |
| Day 4 | 注册了秘塔和飞书，存了模板 | 工具配好 |
| Day 5 | 又写了 2 次周报，发现"风险提示"部分总要改 | 模板 v3 |
| Day 6 | 建了知识库，写了 1 条复盘 | 知识库 + 复盘 |
| Day 7 | 复盘后选了"竞品分析"作为第 2 个任务 | 复盘记录 |

小张的感受：

> "第一天觉得很简单，第三天开始有感觉了——模板改了 3 版之后，周报从 40 分钟降到 12 分钟。最大的收获不是省时间，是知道了 AI 哪里靠谱、哪里不靠谱。"

## 如果你每天只有 15 分钟

缩减版计划：

| 天 | 做什么 |
| --- | --- |
| Day 1 | 列任务 + 选 1 个 |
| Day 2 | 写 1 个模板 + 跑 1 次 |
| Day 3 | 用模板做 1 次真实交付 |
| Day 4 | 注册工具 + 存模板 |
| Day 5 | 再跑 1 次 + 改模板 |
| Day 6 | 建知识库目录 |
| Day 7 | 写复盘 |

关键不是每天做多少，是 7 天都在做真实动作。

## 一周结束的最低标准

| 成果 | 数量 |
| --- | --- |
| 真实任务交付 | ≥ 3 次 |
| 可复用模板 | ≥ 2 个 |
| 知识库 | 已建好目录 |
| 失败复盘 | ≥ 1 条 |
| 下周计划 | 已选定第 2 个任务 |

拿到这些再去学模型选型和智能体，效率会高很多。
